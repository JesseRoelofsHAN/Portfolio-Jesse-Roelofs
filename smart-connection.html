Mijn ervaringen Smart Connection

"Ik wil de basis leren van een website en deze zelf kunnen ontwikkelen"

Tijdens Smart Connection activiteiten heb ik gewerkt aan mijn programeer skill's van planten systeem tot website. Ik ga hierbij in op de weg naar mijn portfolio website. Op deze pagina vindt u een beknopte uiteenzetting van mijn ervaringen gedurende deze module.

Op een rijtje

  • Workshop HTML/CSS

    Tijdens deze workshop werden HTML & CSS geïntroduceerd. Ik had vooraf Microsoft Visual Code en Git geïnstalleerd. Ik opende de Hello World! pagina en leek iets te begrijpen van de code. Vervolgens gingen we bestaande sites inspecten, dit was complexer.

  • Workshop eigen website

    In deze workshop zijn we op basis van de hello world pagina begonnen aan het ontwikkelen van ons eigen portfolio. Ik merkte hierbij dat ik door technische tegenslagen, achter kwam te liggen. Het vervolg van de workshop was ik genoodzaakt mee te kijken met jason, hierdoor leerde ik het zelf niet optimaal. Toch vond ik het erg leuk om te zien hoe het bij Jason op dat moment wel lukte.

  • Workshop databases

    In deze workshop stonden databases centraal, waar komt informatie vandaan en op welke manier? Deze stof is voor mij toch nog iets te hoog gegrepen gebleken. Ik wilde hier zelf graag een toepassing van maken maar dit is mij helaas nog niet gelukt. Wel heb ik een goed beeld gekregen van de opslag van data en de relaties daartussen.

  • Workshop algoritmes

    In deze workshop hebben we een impressie gekregen van algoritmes en AI. Witkek deelde zijn kennis en visie en gaf ons verschillende demonstraties. Hiervan is het Monty Hall Problem mij het meest bijgebleven. Het was heel leerzaam om te zien hoe het algoritme in de praktijk tests uitvoerde en de theorie bevestigde.

Algehele impressie

  • Tijdens de module smart connection heb ik gewerkt aan de ontwikkeling van mijn technische kennis en de ontwikkeling van deze website. Ik had een ander resultaat voor ogen maar zover ben ik nog niet in mijn ontwikkeling, wel heb kan ik zeggen dat ik de basis heb gelegd. Bij deze module had ik achter gezien op tijd meer hulp moeten vragen, ik twijfel er niet aan dat ik dan nog meer had geleerd. Enerzijds kan ik stellen dat ik mijn leerdoel heb behaald, anderzijds vind ik hierbij wel dat ik op dit punt ambitieuzer had mogen zijn.
